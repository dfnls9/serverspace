## Performance: What's Under the Hood

Serverspace runs on two virtualization platforms:

- **vStack** — their hyperconverged platform, used for the top-tier cloud VMs, with Intel Xeon Scalable v2 CPUs and NVMe SSD storage reaching up to 30,000 IOPS
- **VMware** — enterprise-grade VMware ESXi with DRS and High Availability, which automatically reroutes workloads during hardware failures

The hardware story is genuinely good. If a physical host fails, your VM reboots automatically on an alternative host. That's how you deliver a 99.9% SLA uptime guarantee without it being a marketing fantasy.

Deployment speed is also real — Serverspace keeps VMs in a dedicated pre-provisioned pool, so your server goes from configured to running in roughly **40 seconds**.
